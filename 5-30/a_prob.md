# A. 烦恼的一次函数

### 题目描述

`LZ`的同学们一定被最近`hy`的一次函数弄自闭了吧

众所周知，`*ys`是一个爱偷懒的小伙子，~~并且很菜~~

所以，他想让你帮他写一个代码，来求两个一次函数的交点

### 输入格式

读入共一行,共两个字符串，用空格隔开$y=k_1x+b_1$ $y=k_2x+b_2$,其中$k_1,b_1,k_2,b_2$为变量

### 输出格式

共一行,以$(x,y)$的格式输出，x,y保留`5`位小数

### 样例

##### input

```txt
y=2x+1 y=3x+1
```

##### output

```txt
(0.00000,1.00000)
```

##### input

```txt
y=5x+10 y=12x+200
```

##### output

```txt
(-27.14286,-125.71429)
```
### 数据范围&注意
$1≤k_1,b_1,k_2,b_2≤10^5且为整数$
$时间限制 1000ms$
$空间限制 128mb$
